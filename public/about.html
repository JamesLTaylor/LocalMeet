About WareToMeet

Connecting neighbours and communities — simple, private, and ad-free.

A tiny step to make the internet work to make our lives better, just like we hoped it would back in the early days of the web.

What is WareToMeet?

WareToMeet is a small community-focused events site for local groups and neighbours to discover, share, and organise meetups. We prioritise privacy, simplicity and accessibility — no ads, no tracking, and no recommendation algorithms.

Our Principles

  • Community first — built for people organising around shared interests.
  • Privacy by design — we only store the minimum information required.
  • Open and transparent — the project is lightweight and easy to host locally.

How to contribute

If you'd like to contribute events, sign up and add events via the "Add Event" option in the menu. For code contributions, see the repository on GitHub and open a pull request.

Contact & Data

If you need your data removed or have questions, email admin@waretomeet.co.uk.

Other libraries and components we use:

We use a variety of open-source libraries and try to avoid anything from the big tech companies that control so much of the internet:

  • FullCalendar - an open-source JavaScript calendar library for event display.
  • Tom Select - a flexible select box enhancement library for better dropdowns.
  • Icons8 - free icons used in the interface.
  • Postmark - transactional email API for sending event notifications.
  • Get The Data - API for UK postcode lookup and geolocation.
  • OpenStreetMap - open-source mapping data for location services.
